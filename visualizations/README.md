# Visualizations

This folder contains all charts and plots generated during the Univariate Analysis EDA.

## Contents

| Prefix | Dataset | Type |
|--------|---------|------|
| `01_tips_` | Tips Dataset (Beginner) | Numerical & Categorical plots |
| `02_penguins_` | Penguins Dataset (Intermediate) | Numerical, Categorical & Bimodal investigation |
| `03_diamonds_` | Diamonds Dataset (Advanced) | Numerical, Categorical & Transformation comparison |

## Chart Types Included
- Histogram + KDE
- Box Plot
- Violin Plot
- ECDF (Empirical Cumulative Distribution)
- Bar Chart (Categorical frequency)
- Donut Chart
- Bimodal investigation (Overall vs. By-Species)
- Transformation comparison (Original vs. Log vs. √)
